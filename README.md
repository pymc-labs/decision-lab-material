# Decision Lab Material

Marketing materials, demos, and video assets for [Decision Lab](https://github.com/pymc-labs/decision-lab).

## Contents

### `video/`
Remotion-based video demo for Decision Lab — animated walkthrough of the platform.

### `viewer/`
Interactive web UI for viewing Decision Lab experiment sessions as directed graphs. Includes an Express server, single-page app, demo session data, and Fly.io deployment config.
